## EndPoints

List all Beers

```
http://127.0.0.1:8000/beers/search
```

List Beers by food

```
http://127.0.0.1:8000/beers/search?food=Fresh
```

View Beer

```
http://127.0.0.1:8000/beers/view/{id}
```
